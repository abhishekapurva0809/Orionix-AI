# 🚀 Orionix AI

### Modern AI Chat Assistant Platform

Orionix AI is a modern full-stack AI chat assistant project currently under development using React, Vite, Node.js, Express, and Zustand.

The goal of this project is to build a clean, responsive, scalable, and modern conversational AI web application inspired by modern AI assistant platforms.

This project is focused on learning:
- Full Stack Development
- Frontend Architecture
- Backend API Development
- State Management
- Modern UI/UX Design
- AI Chat Application Structure
